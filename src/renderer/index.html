Troubleshooting Steps
1
Check that Ollama is installed on your system. Download it from ollama.com if not.
2
Ensure no other Ollama instance is running. Try quitting Ollama from the menu bar, then relaunch AI Organizer.
3
Check available RAM — the AI model needs at least 4 GB of free memory. Close heavy applications and retry.
4
If the issue persists, try running ollama serve in Terminal to see detailed error logs.

Privacy Policy

Last updated: February 2026

AI Organizer (System Janitor) is designed with privacy as a core principle. Here is how your data is handled:

Local Processing
All file classification and AI processing happens entirely on your device using a bundled AI model (Ollama). No file contents are ever uploaded to any server.

Data We Collect
License validation requires your license key to be verified against our server. No file names, contents, or personal data are transmitted during this process.

External API Calls
The Semantic Knowledge feature sends only category folder names (not file contents) to Datamuse and Wikipedia APIs to fetch related keywords. This is optional and user-initiated.

Cloud Sync
If enabled, files are copied to your own cloud storage (Google Drive or iCloud) via local sync folders. We never access your cloud accounts directly.

No Telemetry
We do not collect usage analytics, crash reports, or any telemetry data. The app operates fully offline except for license validation and optional knowledge expansion.

Data Storage
All settings, learning data, and audit logs are stored locally on your device in the application data directory. You can delete them at any time via Settings.

For questions, contact: privacy@systemjanitor.app
