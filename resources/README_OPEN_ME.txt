HOW TO OPEN AI ORGANIZER
─────────────────────────────────────────────────────────

This app is not signed with an Apple Developer certificate.
macOS will block it on first launch. Follow these steps:

  STEP 1 — Drag "System-Janitor.app" to your Applications folder.

  STEP 2 — DO NOT double-click the app yet.

  STEP 3 — Right-click (or Control+click) the app icon.

  STEP 4 — Select "Open" from the context menu.

            ┌──────────────────────────────────┐
            │    System-Janitor.app            │
            │                                  │
            │  "System-Janitor" can't be       │
            │  opened because it is from an    │
            │  unidentified developer.         │
            │                                  │
            │  ┌────────┐   ┌───────────────┐  │
            │  │  Open  │   │    Cancel     │  │
            │  └────────┘   └───────────────┘  │
            └──────────────────────────────────┘

  STEP 5 — Click "Open" in the dialog that appears.

You only need to do this ONCE. After that, the app opens normally.

─────────────────────────────────────────────────────────
ALTERNATIVE (Terminal method):

  1. Open Terminal (Applications → Utilities → Terminal)
  2. Run this command:
       xattr -cr /Applications/System-Janitor.app
  3. Double-click the app normally.
